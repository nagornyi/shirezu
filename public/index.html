How it works

1. Global Administrator creates a new Company and invites a Company Administrator.

2. Then Company Administrator creates a new Project (multiple projects can be created) and invites a Project Administrator.

3. Then Project Administrator creates new resources and invites Operators to the project.

4. Users can Release/Occupy resources acording to their permissions (Operators cannot release resources occupied by other users, but Project and Company Administrators can do it).

5. The exact time when resource was occupied is displayed on the dashboard, each user sees this time in his own timezone.

6. All users can communicate with each other to discuss the status of any given resource.
